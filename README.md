# ONE EIGHT

ブラウザで動作する ONE EIGHT の対戦実装です。

---

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

---

## テスト

```bash
npm run test
```

---

## 操作手順

### 1. Position を選択する

盤面中央のグリッドにある A〜M のボタンをクリックします。  
選択すると、関連する Gate がハイライトされます。

### 2. Build を実行する

Gate のポケット（菱形ピップ）をクリックします。

| ポケット | Build タイプ | 操作 |
|---|---|---|
| Large（大） | Massive | クリック1回で即実行 |
| Middle（中） | Selective | 1つ目の Gate → 2つ目の Gate の順にクリック |
| Small（小） | Quad | 4つの Gate を順にクリック（4つ目で実行） |

- 再クリックで選択解除できます（Selective 1つ目 / Quad 各Gate）。
- 現在の操作状態は「Build」パネルのヒントに表示されます。

### 3. Pass（スキップ）

Build できる手がない場合のみ「Pass」ボタンが有効になります。

### 4. New Game

ヘッダーの「New Game」ボタンで新しいゲームを開始します。  
ゲーム終了後の「Result」パネルからも起動できます。

### 5. Undo

「Undo」ボタンで直前の手を1手戻せます。  
CPU 対戦時は、直前の人間の手まで遡ります。

### 6. 棋譜のコピー

- 「Move History」パネルの「Copy record」ボタン  
- または終局後「Result」パネルの「Copy record」ボタン  
でクリップボードにコピーできます。

### 7. 棋譜のインポート

「Import record」パネルにコピーした棋譜を貼り付けて「Import record」ボタンを押すと、  
その局面から再開できます。

---

## 保存 / 再開

- 各手を指すたびに自動でブラウザのローカルストレージに保存されます。
- 次回アクセス時、前回の状態が自動で復元されます。
- ヘッダーに「Saved」バッジが表示されているとき、「Clear save」で保存データを削除できます。

---

## 棋譜フォーマット

```
1. A, m(3)
2. B, s(1,4)
3. C, q
4. P
```

- `m(N)` — Massive（Gate N）
- `s(N1,N2)` — Selective（Gate N1 と N2）
- `q` — Quad
- `P` — Pass
