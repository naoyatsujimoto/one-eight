## 考察

### 1. 初手分布の均一性
- 全13種のpositioningが約7.5〜7.9%とほぼ均等に出現
- Easy CPUは初手に強い偏りを示さない（ランダム性が高い）
- L(7.86%) が最頻出だが、B(7.53%) との差は0.33%以内

### 2. 初手別 black勝率の格差
- **最強初手: G（57.60%）** — 他と比べて突出した黒有利
- **最弱初手: H（48.57%）** — 黒が不利（白勝率 51.3%超）
- G と H の差は約9ポイントと有意な差がある
- G, M, K, D, J, I, C, A が50%超（black有利グループ）
- H, L, F が50%未満（white有利グループ）
- B がほぼ均衡（50.11%）

### 3. build_type × positioning の交互作用
- **G + massive が最強（61.49%）** — G positioningで massiveを使うと黒勝率が最も高い
- **H + massive が最弱（48.21%）** — 同じ massive でも positioning による差が大きい
- 全体的に massive は selective より勝率が高い傾向
- quad は selective と massive の中間程度の効果

### 4. Easy CPU の特性への示唆
- 初手選択が均等なため、初手段階でのEasy CPU特有の「癖」は小さい
- ただし G positioning で始まる局はblackが有意に有利 → G positioning の何らかの構造的優位性が示唆される
- H, L, F で始まる局はwhiteが有利 → これらのpositioningは先手（black）に不利な可能性
